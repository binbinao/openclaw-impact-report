## 执行摘要

大模型市场正处于从实验阶段向规模化部署的关键转折点。本报告深入分析了 Tokens 计费与 Coding Plan（订阅制）两种核心商业模式的差异、优劣势及适用场景。调研发现：

**市场规模**：AI市场预计从2024年1840亿美元增长至2030年8260亿美元（年复合增长29%），但仅39%企业报告AI对EBIT产生影响。

**价格竞争**：字节豆包以¥0.5/百万tokens成为国内最低价，国际厂商GPT-4o-mini以$0.15/百万tokens提供高性价比选择。

**模式分化**：国际厂商重订阅+C端（ChatGPT Plus $20/月），国内厂商重API+B端（企业私有化部署成标配）。

**投资机会**：集中在企业级AI应用、Agent自动化、垂直行业解决方案、AI基础设施和多模态应用五大领域。

## 一、Tokens 计费模式深度分析

### 1.1 Token 的技术定义与计算原理

**Token 的本质**

Token 是 LLM 处理文本的基本单位，模型不直接处理原始文本，而是将文本转换为数字序列（token IDs）。根据 OpenAI 官方定义，token 通常对应 4 个字节的文本，或约 0.75 个英文单词。

Token 的核心特性：
- **可逆性：** 通过 tokenizer 的 decode 方法可将 token IDs 完整还原为原始文本
- **压缩性：** Token 序列长度通常短于原始文本的字节长度
- **语言依赖性：** 不同语言的 token 效率差异显著（英文约 1 token = 4 字符，中文约 1 token = 1-2 汉字）

**主流 Tokenization 算法**

| 算法 | 代表模型 | 词汇表大小 | 特点 |
|------|----------|-----------|------|
| **BPE** | GPT系列、Llama、Qwen2 | 50K-200K | 高效处理常见子词，计算效率高 |
| **Unigram** | T5、BigBird、Pegasus | ~100K | 概率方法，多语言适应性强 |
| **WordPiece** | BERT系列 | ~30K | 最大化训练数据似然 |

**OpenAI Tokenizer 演进**

| Encoding | 模型 | 词汇表大小 | 多语言优化 |
|----------|------|-----------|-----------|
| o200k_base | GPT-4o | ~200K | 显著提升 |
| cl100k_base | GPT-4 | ~100K | 中等 |
| p50k_base | GPT-3 | ~50K | 基础 |

**多语言 Token 效率差异**

相同文本在不同 tokenizer 下的 token 数量差异可达 75%：

| 语言 | GPT-4o (o200k) | GPT-4 (cl100k) | GPT-3 (p50k) |
|------|---------------|---------------|-------------|
| 英文 | 基准 | +0-10% | +10-20% |
| 中文 | 基准 | +20-30% | +50-80% |
| 日文 | 基准 | +12% | +75% |

### 1.2 Tokens 计费的优缺点

**核心优势**

| 优势 | 说明 |
|------|------|
| ✅ 精确成本归因 | 细粒度成本追踪，公平反映资源消耗 |
| ✅ 透明定价结构 | "每百万token"单位便于横向比较 |
| ✅ 激励高效使用 | 推动开发者优化prompt设计 |

**主要缺点**

| 缺点 | 说明 |
|------|------|
| ❌ Token计算不透明 | 普通用户难以直观理解token数量 |
| ❌ 输入/输出价格差异 | 输出价格是输入的3-5倍，长输出任务成本高 |
| ❌ 上下文长度成本压力 | 200K上下文输入成本可达$30+ |
| ❌ 多语言不公平性 | 中文成本比英文高30-50% |

### 1.3 成本优化策略

**Prompt Caching（革命性技术）**

| 场景 | 无缓存成本 | 有缓存成本 | 节省比例 |
|------|-----------|-----------|---------|
| 100次相同系统prompt | $7.50 | $5.25 | 57% |
| 1000次相同系统prompt | $75.00 | $52.50 | 90% |

**批量处理折扣**

| 模型 | 实时API价格 | 批量API价格 | 折扣比例 |
|------|------------|------------|---------|
| GPT-4o | $2.50/$10.00 | $1.25/$5.00 | 50% |
| GPT-4o-mini | $0.15/$0.60 | $0.075/$0.30 | 50% |

**Token精简策略**

| 优化方法 | Token减少比例 | 实施难度 |
|---------|-------------|---------|
| 移除冗余词汇 | 40% | 低 |
| 使用符号和缩写 | 30% | 低 |
| 合并相似指令 | 50% | 中 |
| 结构化Prompt（JSON/XML） | 20-40% | 中 |

**实际案例：企业知识库问答系统**

| 方案 | 月成本 | 准确率 |
|------|-------|--------|
| 完整上下文 | $750,000 | N/A |
| 传统RAG | $3,750 | 60% |
| Contextual Retrieval + Caching | $1,500 | 95% |

## 六、核心发现与建议

### 6.1 核心发现

**1. 价格竞争白热化**
- 字节豆包以¥0.5/百万tokens成为国内最低价
- 所有厂商都提供免费模型或试用额度
- 2024-2025年价格下降幅度超过50%

**2. 商业模式分化明显**
- 国际厂商：订阅制+API双轨制，注重C端变现
- 国内厂商：以API计费为主，注重B端服务
- 开源厂商：Meta Llama完全免费，通过生态变现

**3. Tokens计费技术驱动**
- Token效率因语言和tokenizer差异可达20-40%
- 输入/输出价格比约1:3-1:5
- Prompt Caching可降低成本50-90%

**4. 企业服务成为标配**
- 所有厂商都提供企业级方案
- 私有化部署成为企业服务标配
- 定制化训练服务普遍提供

**5. 投资机会集中五大领域**
- 企业级AI应用、Agent自动化、垂直行业解决方案、AI基础设施、多模态应用
